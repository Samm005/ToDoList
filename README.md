## To-Do list application using React

This To-Do List application is developed using React and Vite.
It enables users to perform several functions including adding, editing, deleting, and marking tasks as completed.
The application integrates localStorage, allowing it to persist user data even after page reloads, thereby enhancing the user experience by maintaining task information between sessions.

## Features

- Add new to-do tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Persist tasks using browser localStorage
- Clean and user-friendly UI

## Technology used

- React
- Vite
- Local storage
- CSS
- Javascript

## How to run the project

1. Clone the repository(if using git and github) or extract files (if using zip file)

2. Navigate to the todo-list folder
   cd todo-list

3. Install dependencies (npm uses package-lock.json)
   npm install

4. Run the application
   npm run dev

5. Open browser and go to the link provided by vite
